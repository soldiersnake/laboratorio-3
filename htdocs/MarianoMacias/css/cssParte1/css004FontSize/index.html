Tamaño de los fonts

Los textos marcados con "H1" tendran dimension doble a los parrafos comunes. Corresponderian a titulos.

En el ejercicio se definira un tamaño de letra default para el body y luego todo sera expresado en unidades em (unidad de referencia)

En este ejercicio se trabajara con control de medio @media() para que el tamaño de letra sea modificado segun el tamaño del viewport

Dispositivos moviles de alta resolucion tienen una definicion mayor a: 300ppi (pixels/pulgadas)

Dispositivos moviles de baja resolucion tienen una definicion menor en el orden de los 100ppi (pixeles/pulgadas)

Dispositivos de pantalla grande tienen una definicion en el orden de los 60 (pixeles/pulgadas)

En un movil de alta resolucion estaria viendo las letras de 3 a 5 veces mas chicas qe en una pantalla grande si su tamaño es fijo en pixeles

Por este motivo es que si no uso el control de medio, sera mas conveniente definir el tamaño de letra considerando el peor de los casos (los dispositivos de alta resolucion)

En este ejercicio hago crecer la letra conforme el dispositivo es mas pequeño, siempre suponiendo que estos tienen mayor resolucion

12px (1rem) para dispositivos mayores a 1080px. No olvidar definir el tamaño default en 12px

24px (2em) para dispositivos mayores a entre 800px y 1080px

26px (3rem) para dispositivos menores a 800px
